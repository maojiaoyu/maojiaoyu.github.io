## 数理与电化学基石

目标：基石，这是基石！盘古开天辟地，一锅有机粥，电光火石间迸发的第一个灵感，亿万年来的打打杀杀，方有这数百年的精华字节。站在前人的肩膀上，站在有机生物的顶端，让你建立第一性原理思维，能看懂文献公式，能解析测试数据。

### 数学与计算基础

- [ ] 统计学：掌握实验设计和实验数据分析，如DOE、误差传递等
- [ ] 微积分与线性代数：掌握偏微分方程（PDE）基础，理解扩散方程的推导
- [ ] Python编程基础：熟练使用 NumPy, Pandas, Matplotlib/Seaborn
- [ ] 电池数据处理实战：
- [ ] 编写脚本读取 Arbin/BioLogic/Neware 原始数据文件
- [ ] 实现充放电曲线绘制、容量标定、库伦效率计算
- [ ] 实现 dQ/dV (增量容量分析) 和 dV/dQ 曲线计算与平滑处理
- [ ] 使用 scipy.optimize 进行电化学模型参数拟合
- [ ] MATLAB/Simulink基础（可选，BMS方向必学）：掌握 Simscape Electrical 电池模块

### 物理化学与电化学核心

- [ ] 热力学基础：
	- [ ] 标准电极电位
	- [ ] 吉布斯自由能与开路电压(OCV)的关系 ($\Delta G = -nFE$)
	- [ ] Nernst方程推导与应用
	- [ ] 锂离子嵌入反应的热力学特征
	- [ ] 活度系数与浓度过电位

- [ ] 动力学基础：
	- [ ] Butler-Volmer 方程的物理意义与Tafel近似
	- [ ] Fick扩散定律（稳态与非稳态）
	- [ ] 电荷转移阻抗 vs 扩散阻抗 vs 欧姆阻抗的区别
	- [ ] 锂离子在固相（电极材料）和液相（电解液）中的传输机制

- [ ] 界面电化学：
	- [ ] 双电层模型 (Helmholtz / Gouy-Chapman / Stern)
	- [ ] SEI膜的形成机理、组分与生长动力学
	- [ ] CEI膜与正极界面副反应

- [ ] 电化学测试方法：
	- [ ] 循环伏安法(CV)：峰位偏移、峰电流与扫速关系(Randles-Sevcik)
	- [ ] 电化学阻抗谱(EIS)：等效电路拟合、弛豫时间分布(DRT)分析
	- [ ] 恒流间歇滴定(GITT/PITT)：固相扩散系数测量
	- [ ] 漏电流与自放电测试原理

### 材料表征技术

- [ ] 结构表征：XRD (Rietveld精修)、中子衍射(原位锂占位)
- [ ] 形貌表征：SEM/EDS、TEM/SAED、FIB-SEM三维重构
- [ ] 表面与价态：XPS、AES、TOF-SIMS深度剖析
- [ ] 热分析：DSC/TGA、ARC绝热加速量热仪原理与数据解读
- [ ] 原位/工况表征：In-situ XRD/Raman/TEM 实验设计与数据关联

## 材料体系与电芯设计

目标：修炼到大成，可以知道什么是电池和电池怎么做了！掌握主流体系优缺点，具备独立设计电芯和分析失效的能力。

### 正极材料体系

- [ ] 钴酸锂 (LCO)：晶体结构、高压应用挑战、改性策略
- [ ] 磷酸铁锂 (LFP)：橄榄石结构、两相反应机制、碳包覆/离子掺杂、低温性能改善
- [ ] 三元材料 (NCM/NCA)：

- [ ] 层状结构与阳离子混排
- [ ] Ni含量对容量/稳定性/安全性的权衡
- [ ] 单晶化技术与多晶颗粒破碎问题
- [ ] 高镍材料的表面残碱处理与湿法包覆

- [ ] 富锂锰基 / 高压尖晶石：氧参与氧化还原机制、电压衰减问题
- [ ] 钠离子正极：层状氧化物、普鲁士蓝类、聚阴离子化合物对比

### 负极材料体系

- [ ] 石墨：储锂机制(阶化合物)、各向异性、快充析锂边界
- [ ] 硅基负极：体积膨胀(~300%)、SEI反复破裂再生、纳米化/多孔化/CVD包覆策略
- [ ] 硬碳/软碳：储钠机制(吸附+填充)、首效提升、平台区调控
- [ ] 锂金属负极：枝晶生长模型、固态电解质界面(SSEI)、亲锂骨架设计
- [ ] 钛酸锂 (LTO)：零应变特性、胀气机理与抑制

### 电解液

- [ ] 溶剂体系：EC/EMC/DEC等碳酸酯物性、氟代溶剂、砜类/腈类高压溶剂
- [ ] 锂盐：LiPF₆水解敏感性、LiFSI/LiTFSI优势与铝箔腐蚀问题
- [ ] 功能添加剂：成膜添加剂(VC/FEC)、阻燃添加剂、过充保护、HF捕获剂
- [ ] 新型电解质：局部高浓(LHCE)、离子液体、聚合物/硫化物/氧化物固态电解质
- [ ] 隔膜：PP/PE干法vs湿法、陶瓷涂覆、芳纶涂覆、耐热收缩性能

### 隔膜

- [ ] 聚烯烃隔膜的结构、闭孔特性与陶瓷涂层

### 粘结剂

- [ ] PVDF/SBR等粘结剂的作用机理与选型

### 电芯设计与工艺

- [ ] 电化学设计：
	- [ ] N/P比计算与安全窗口设定
	- [ ] 面容量、压实密度、孔隙率、曲折度的匹配
	- [ ] 电解液注液量与浸润时间估算
- [ ] 结构设计：卷绕vs叠片、极耳位置优化、防爆阀设计、CTP/CTC集成
- [ ] 制造工艺关键控制点：

	- [x] [涂布](https://blog.maojiaoyu.com/blog/lithium-ion-battery-coating-process)：干燥速率梯度、边缘效应、双面一致性 ✅ 2026-07-06
	- [x] [辊压](https://blog.maojiaoyu.com/blog/lithium-ion-battery-rolling-process)：回弹控制、极片波浪边、粘结剂迁移 ✅ 2026-07-06
	- [x] [装配](https://blog.maojiaoyu.com/blog/lithium-ion-battery-assembly-process)：水分控制(ppm级)、毛刺检测、焊接质量 ✅ 2026-07-06

## 系统集成与工程应用

目标：做大做强，再创辉煌！从单体走向系统，掌握BMS、热管理与仿真能力。

### 电池建模与仿真

- [ ] 等效电路模型(ECM)：Thevenin模型、DP模型、PNGV模型、参数辨识(HPPC/EIS)
- [ ] 电化学机理模型(P2D)：COMSOL/PyBaMM搭建、简化模型(SPMe/TSPMe)
- [ ] 热仿真：CFD流体仿真、液冷板设计优化、相变材料散热
- [ ] 机械仿真：膨胀力建模、模组结构强度、振动疲劳分析
- [ ] AI+Battery：GPR/LSTM寿命预测、神经网络替代机理模型、图像缺陷检测

### BMS算法与策略

- [ ] 状态估计：

- [ ] SOC：安时积分+OCV校正+EKF/UKF/AEKF
- [ ] SOH：容量标定、内阻跟踪、IC/DV特征提取、数据驱动模型
- [ ] SOP：基于温度/SOC/老化状态的功率边界动态查表

- [ ] 均衡管理：被动均衡vs主动均衡拓扑、均衡策略与时机选择
- [ ] 充电策略：恒流恒压(CC-CV)优化、脉冲充电、低温加热充电、快充MAP标定
- [ ] 故障诊断：内短路早期预警、传感器故障识别、绝缘检测、热失控分级报警
- [ ] 云端BMS：大数据平台架构、数字孪生、OTA远程标定
